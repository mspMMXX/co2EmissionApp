# CO2 Emission App

## Beschreibung
Diese Anwendung ermöglicht es, den aktuellen CO2-Ausstoß eines Landes anzuzeigen und zu verwalten.

## Technologien
- Backend: Spring Boot (Java)
- Datenbank: MySQL (Docker)
- Frontend: React (geplant)

## Funktionen
- Anzeige des CO2-Ausstoßes eines Landes
- Registrierung und Login von Benutzern
- Hinzufügen und Aktualisieren von Emissionsdaten

## Projektstruktur
- backend: Spring Boot Anwendung
- frontend: React Anwendung

## Starten des Backends
```bash
cd backend
./mvnw spring-boot:run
